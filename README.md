Gatsby transformer plugin for jupyter files.

## Usage

- Install: `yarn add gatsby-transformer-ipynb`
- Add to your plugins in `gatsby-config.js`:
  ```js
  plugins: ['gatsby-transformer-ipynb']
  ```
